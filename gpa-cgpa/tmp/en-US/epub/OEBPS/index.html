Abstract
The User Documentation serves the purpose to guide the user's of GPA and CGPA calculator. GPA refers to the Grade Point Average where the CGPA is Cumulative Grade Point Average. Through the guide, the user shall be able to learn how to use the program or in some scenario the desktop software concurrently.
